![图1 原始数据](./image/original_data.png)

由8列特征组成，分别代表体温、恶心、腰疼、连续排尿、排尿疼痛、尿道肿胀、膀胱炎症和肾炎。除了体温为数值属性，其他属性都是二值的标称属性，所以考虑将体温转换为同为标称属性的发烧属性。在实验中选取38摄氏度为基准，低于38度为正常，高于38度为发烧。
